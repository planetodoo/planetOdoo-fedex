Shipping Services Management With Odoo

Shipping Services Management Module Provides you with Integration of shipping services like FedEx,UPS and USPS in Odoo



Setup & Configure

                 
After installing the module You will find the configuration under "Inventory", named as "Shipping Service Management" Here the configuration for the required shipping service can be done, so here we make the configuration for FEDex
Under "Inventory" >> "Shipping services Management", We have USPS setting where we deifine USPS setting to generate shipping Rates and Endicia Setting to generate shipping labels, we have option to set the format for label aswell.
Under "sales" there is configuration for shop, where we need to configure a shop which will be used as shop for that shipping service

Category & Product Based Shipping Configuration

Under "Sales" >> "Products" >> "Category, There is option to select the type of shipping service we want to use for the products that belongs to this category
Under "Sales" >> "Product Variant", Under "Inventory" Tab we select the category for product, and if category is configured for shipping method, then that shipping service is applied to that product.
There is also option available to configure shipping service at product level So under "Sales" >> "Product Variant" There is tab Called "Shipping Method" where user can set the type of shipping service needed For that Product

Generate Shipping Quotes In Sales Order

Under sales >> "Sales Order", here we need to select the shop, appropriate to the service we going to use to ship the corresponding order
Here we have Shipping service Related fields , We have the "Shipping type" where you can select the type of shipping service, you want to ship the corresponding order
In Sale order we have a tab called "Shipping Info" where we can select the type of shipping service, the weight of package, type of packaging and based on these information we can generate the shipping rate of that parcel to that corresponding service by clicking on "Generate Shipping Quotes", and the generated Information is Displayed in the table below.

Generate Shipping Quotes In Delivery Order

Under "Inventory" >> "All Transfers", We have delivery Order which also has the option to generate shipping rates, select the type of shipping service and then by clicking on "Generate Shipping Quotes" we get the Service specific Rate information in the below tabel

Generate Shipping Labels In Delivery Order

     
In order to Generate shipping labels after we have the shipping service selected in delivery order by means of product category configuration or product configuration or self editing, Under "Inventory" >> "All Transfers", we have the option to select the order for which we want to generate shipping label and click on "Generate Label"
As we select the Generate label option there a batch number that appears in the pop up, and this batch number gets saved in individual delivery order for which the shipping label was generate together.
After the Generate Label batch run is done we get the individual order specific label in each delivery order In the attachments, and we have the "Label Generated" checkbox Checked. As we select the Generate label option there a batch number that appears in the pop up, and this batch number gets saved in individual delivery order for which the shipping label was generate together.
After the label is generated we get the carrier and the tracking number Here
